BCA(Bachelor in Computer Application)

BCA, full form Bachelor of Computer Application, is a 3-year UG course that imparts knowledge on the basics of computer application and software development. BCA is almost equivalent to the BTech course. The major benefit of opting for BCA is that it is a 3-year program whereas BTech is a 4-year course. A Studnet with a BCA degree is in huge demand since manpower is continuing to get replaced by the systems and the need for good skillful experts and professionals is there to operate and function.

Eligibility

The student must have an aggregate of 50% or above in Class 12 in any stream (Arts, Science, Commerce) from a any board.

Subject you will be studying.

  • Foundational Mathematics
  • Digital Computer Fundamentals
  • Data Structures
  • Financial Accounting
  • Programming In Java
  • Multimedia Applications
  • ETC.

Jobs Opportuinites

  • Software Developer
  • Network Engineer
  • Software Architect
  • Website Designing
  • IT Department
  • Mobile App Development
  • ETC.
